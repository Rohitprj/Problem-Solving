TAILWIND CSS CHEAT SHEET

--------------------------------------------------
1) TEXT PROPERTIES IN CSS → TAILWIND
--------------------------------------------------

Color:
<p class="text-red-500">This text is red</p>
<p class="text-blue-600">Blue text</p>
<p class="text-green-500">Green text</p>
<p class="text-black">Black text</p>

Text Alignment:
<p class="text-left">Left aligned</p>
<p class="text-center">Center aligned</p>
<p class="text-right">Right aligned</p>

Text Decoration:
<p class="underline">Underlined text</p>
<p class="overline">Overlined text</p>
<p class="line-through">Strikethrough text</p>
<p class="no-underline">No underline</p>

Decoration Styles:
<p class="underline decoration-dotted">Dotted underline</p>
<p class="underline decoration-dashed">Dashed underline</p>
<p class="underline decoration-double">Double underline</p>
<p class="underline decoration-wavy">Wavy underline</p>

Font Weight:
<p class="font-normal">Normal</p>
<p class="font-bold">Bold</p>
<p class="font-light">Lighter</p>
<p class="font-extrabold">Bolder</p>

Font Weight Scale:
100 → font-thin
200 → font-extralight
300 → font-light
400 → font-normal
500 → font-medium
600 → font-semibold
700 → font-bold
800 → font-extrabold
900 → font-black

Font Family:
<p class="font-sans">Sans font</p>
<p class="[font-family:Arial]">Arial text</p>
<p class="[font-family:Arial,Roboto,sans-serif]">Fallback fonts</p>

Text Transform:
<p class="uppercase">UPPERCASE</p>
<p class="lowercase">lowercase</p>
<p class="capitalize">Hello World</p>
<p class="normal-case">Normal</p>

--------------------------------------------------
18) QUICK MAPPING
--------------------------------------------------

color: red → text-red-500
text-align: center → text-center
font-weight: bold → font-bold
font-family: Arial → [font-family:Arial]
text-transform: uppercase → uppercase
border → border-2 border-black
padding → p-5
margin → m-5
display: none → hidden
visibility: hidden → invisible
position: relative → relative
position: absolute → absolute
position: fixed → fixed
justify-content: center → justify-center
align-items: center → items-center
transition → transition-all duration-300
