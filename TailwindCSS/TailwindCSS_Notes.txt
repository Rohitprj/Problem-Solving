Tailwind CSS Cheat Sheet 1) Text Properties in CSS → Tailwind color: red Use
text color utilities.

<p class="text-red-500">This text is red</p>

Common examples:

<p class="text-blue-600">Blue text</p>
<p class="text-green-500">Green text</p>
<p class="text-black">Black text</p>
text-align: left / right / center
<p class="text-left">Left aligned</p>
<p class="text-center">Center aligned</p>
<p class="text-right">Right aligned</p>
text-decoration Underline
<p class="underline">Underlined text</p>
Overline
<p class="overline">Overlined text</p>
Line-through
<p class="line-through">Strikethrough text</p>
None
<p class="no-underline">No underline</p>
Decoration style: dotted, dashed, double, wavy
<p class="underline decoration-dotted">Dotted underline</p>
<p class="underline decoration-dashed">Dashed underline</p>
<p class="underline decoration-double">Double underline</p>
<p class="underline decoration-wavy">Wavy underline</p>
font-weight Normal / Bold / Lighter / Bolder
<p class="font-normal">Normal</p>
<p class="font-bold">Bold</p>
<p class="font-light">Lighter</p>
<p class="font-extrabold">Bolder</p>
Font weights from 100 to 900
<p class="font-thin">100</p>
<p class="font-extralight">200</p>
<p class="font-light">300</p>
<p class="font-normal">400</p>
<p class="font-medium">500</p>
<p class="font-semibold">600</p>
<p class="font-bold">700</p>
<p class="font-extrabold">800</p>
<p class="font-black">900</p>
font-family: arial Tailwind gives you font family utilities, or you can use
custom values.

<p class="font-sans">Sans font</p>

For exact Arial:

<p class="[font-family:Arial]">Arial text</p>

For fallback fonts:

<p class="[font-family:Arial,Roboto,sans-serif]">
  Arial, then Roboto, then sans-serif
</p>
text-transform
<p class="uppercase">uppercase text</p>
<p class="lowercase">LOWERCASE TEXT</p>
<p class="capitalize">hello world becomes Hello World</p>
<p class="normal-case">normal case</p>
2) Units in CSS → Tailwind Tailwind mostly uses a spacing scale based on rem,
but you can also use arbitrary values. font-size: 2px
<p class="text-[2px]">Tiny text</p>
Width / Height in pixels
<div class="w-[96px] h-[50px] bg-gray-300"></div>
Percentage
<div class="w-1/2">50% width</div>
<div class="w-full">100% width</div>
Viewport units
<div class="h-screen">100vh</div>
<div class="w-screen">100vw</div>
rem-based example
<p class="text-base">1rem</p>
<p class="text-lg">bigger text</p>
Practice Set 1 Q1: Create a heading centred on the page with all text
capitalized by default
<h1 class="text-center uppercase text-3xl font-bold">
  welcome to tailwind css
</h1>
Q2: Set the font family of all content in the document to "Times New Roman" Best
way: apply on body.

<body class="[font-family:'Times_New_Roman',serif]">
  <h1>Hello</h1>
  <p>This page uses Times New Roman.</p>
</body>

If you want the exact fallback style:

<body class="[font-family:'Times_New_Roman',Times,serif]">

  3) Box Model in CSS → Tailwind Height and Width
  <div class="h-40 w-64 bg-blue-200"></div>
  Border Border width + style + color
  <div class="border-2 border-solid border-black"></div>
  Shorthand in Tailwind Tailwind usually combines these like this:

  <div class="border-2 border-black"></div>
  Different border styles
  <div class="border-2 border-dotted border-black"></div>
  <div class="border-2 border-dashed border-black"></div>
  <div class="border-2 border-double border-black"></div>
  Border radius
  <div class="rounded-lg"></div>
  <div class="rounded-full"></div>

  Examples:

  <div class="w-20 h-20 bg-blue-500 rounded-full"></div>
  Padding
  <div class="p-4">Padding all sides</div>
  <div class="px-4 py-2">Horizontal and vertical padding</div>
  <div class="pt-4 pr-2 pb-6 pl-1">Individual sides</div>
  Margin
  <div class="m-4">Margin all sides</div>
  <div class="mx-auto">Auto horizontal margin</div>
  <div class="mt-4 mb-2">Top and bottom margin</div>
  Clockwise spacing idea CSS: padding: top right bottom left; Tailwind
  equivalent:

  <div class="pt-2 pr-4 pb-6 pl-8"></div>

  7) Units in CSS → Tailwind %
  <div class="w-1/2">50% width</div>
  <div class="w-3/4">75% width</div>

  </button>
