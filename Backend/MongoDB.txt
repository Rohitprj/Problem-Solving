1. Show All Databases
   show dbs

   Example Output:-
     admin
     config
     studentDB
     local

   Explain
   This :- command shows all databases available in MongoDB.

2. Create / Switch Database
   use studentDB

   Explain
   If database exists → switches to it
   If not exists → creates it when data is inserted

3. Check Current Database
   db

   Explain
   This :- command shows current database. 

4. Create Collection
   db.createCollection("collectionName")

   Explain
   This :- command creates a collection in current database.

5. Insert One Document
   db.collectionName.insertOne({})

   Example Output:-
     db.students.insertOne({
     name: "Rohit",
     age: 22,
     course: "MERN"
     })

   Explain
   This :- command inserts one document in current collection.

6. Insert Multiple Documents
   db.students.insertMany([
    {
     name: "Amit",
     age: 21,
     course: "React"
    },
    {
    name: "Priya",
    age: 23,
    course: "Node"
    }
 ])

   Explain
   This :- command inserts multiple documents in current collection.

 7. Find All Documents
    db.students.find()

    Explain
    This :- command finds all documents in current collection.

 8. Pretty Output
    db.students.find().pretty()

    Explain   
    This :- command Formats data in readable style.

9. Find One Document
db.students.findOne({ name: "Rohit" })

Explain
Returns first matching document.

10. Filter Documents
Find by age
db.students.find({ age: 22 })

11. Comparison Operators-------------------------------------- 
Greater than
db.students.find({ age: { $gt: 21 } })
Meaning

Find students whose age is greater than 21.

Less than
db.students.find({ age: { $lt: 23 } })

Meaning
Find students whose age is less than 23.

Greater than equal
db.students.find({ age: { $gte: 22 } })

Meaning
Find students whose age is greater than or equal to 22.

Less than equal
db.students.find({ age: { $lte: 22 } })

Meaning
Find students whose age is less than or equal to 22.

12. AND Condition
db.students.find({
  age: 22,
  course: "MERN"
})

Explain
Both conditions must match.   

13. OR Condition
db.students.find({
  $or: [
    { course: "React" },
    { age: 23 }
  ]
})

Explain
Either condition must match.

14. Update One Document
db.students.updateOne(
  { name: "Rohit" },
  {
    $set: {
      age: 25
    }
  }
)
Explain
Updates only one matching document.

------------------------------------------------------------------------------------------------

15. Update Multiple Documents
db.students.updateMany(
  { course: "MERN" },
  {
    $set: {
      status: "Active"
    }
  }
)
Explain
Updates multiple matching documents.

16. Increment Value
db.students.updateOne(
  { name: "Rohit" },
  {
    $inc: {
      age: 1
    }
  }
)
Explain

Increases age by 1.

17. Delete One Document
db.students.deleteOne({
  name: "Amit"
})

Explain
Deletes only one matching document.

18. Delete Multiple Documents
db.students.deleteMany({
  course: "React"
})

Explain
Deletes multiple matching documents.

19. Count Documents
db.students.countDocuments()

Explain
Returns total number of documents.

Count with condition
db.students.countDocuments({
  course: "MERN"
})

Explain
Returns total number of documents that match the condition.

20. Sort Documents
Ascending
db.students.find().sort({ age: 1 })
Explain

1 = ascending

Descending
db.students.find().sort({ age: -1 })
Explain

-1 = descending

21. Limit Results
db.students.find().limit(2)

Explain
Returns only 2 documents.

22. Skip Documents
db.students.find().skip(2)
Explain

Skips first 2 documents.

23. Pagination Example
db.students.find().skip(0).limit(5)

Page 2
db.students.find().skip(5).limit(5)

24. Select Specific Fields
db.students.find(
  {},
  {
    name: 1,
    course: 1
  }
)

Shows only:
     name
     course

Explain
db.collection.find(filter, projection)

First Argument: {} → Filter - “Match all documents”

Second Argument: Projection
{
  name: 1,
  course: 1
}

This controls which fields are returned.

1 means include
0 means exclude

25. Remove Field
db.students.updateOne(
  { name: "Rohit" },
  {
    $unset: {
      status: ""
    }
  }
)

Explain
Removes status field.

26. Create Nested Object
db.students.insertOne({
  name: "Rohit",
  address: {
    city: "Delhi",
    pincode: 110001
  }
})

Explain
Creates a nested object.

27. Query Nested Object
db.students.find({
  "address.city": "Delhi"
})

Explain
Queries a nested object.

28. Array Example
db.students.insertOne({
  name: "Rohit",
  skills: ["React", "Node", "MongoDB"]
})

Explain
Creates an array.

29. Search in Array
db.students.find({
  skills: "React"
})

Explain
Queries an array.

30. Push Into Array
db.students.updateOne(
  { name: "Rohit" },
  {
    $push: {
      skills: "Express"
    }
  }
)

Explain
Pushes a value into an array.

31. Pull From Array
db.students.updateOne(
  { name: "Rohit" },
  {
    $pull: {
      skills: "Node"
    }
  }
)

Explain
Pulls a value from an array.

32. Drop Collection
db.students.drop()

Explain
Deletes entire collection.

33. Drop Database
db.dropDatabase()

Explain
Deletes current database.


Important Operators To Teach

| Operator | Meaning            |
| -------- | ------------------ |
| `$gt`    | greater than       |
| `$lt`    | less than          |
| `$gte`   | greater than equal |
| `$lte`   | less than equal    |
| `$set`   | update field       |
| `$inc`   | increment          |
| `$unset` | remove field       |
| `$push`  | add to array       |
| `$pull`  | remove from array  |
| `$or`    | OR condition       |
