2) Entities & key fields (explanation)
USER

Primary user table with authentication info and profile fields: id, name, email, phone, dob, gender, country, city, password_hash, role_id.

role_id links to ROLE for admin/subadmin/regular user access control.

ROLE

Roles like ADMIN, SUBADMIN, USER. Used for RBAC.

BOOK

Catalog of books (you mentioned a dropdown of 7 books initially; this table stores the catalog).

USER_BOOK is a join table storing which books a user has read and extra metadata (status, rating, notes).

USER_BOOK

Many-to-many between USER and BOOK. Stores read_date, status, rating, notes.

DOCUMENT

For structured PDFs / books / domain data that the chatbot will index. Stores file_url and uploader.

DOCUMENT_TAG

Lightweight tagging for documents to improve retrieval/filtering.

CHAT_SESSION

Every chat session (text/voice/hybrid) the user starts. Tracks start/end time, session type, and optional bot_model used.

MESSAGE

Messages within a session. Supports message_type (text, audio, file, system). For audio we store media_url and duration_seconds. content for transcripts or text bodies.

ATTACHMENT

Files attached to a message (e.g., a transcript file, audio recording, images).

BOT_MODEL

Optional table to keep which model/version served the session (for auditing / experiments).

3) Example PostgreSQL DDL (starter)
Run or adapt as needed — these include FK constraints and sensible columns.

-- roles
CREATE TABLE role (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name VARCHAR(50) NOT NULL UNIQUE,
  description TEXT
);

-- users
CREATE TABLE "user" (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name VARCHAR(200) NOT NULL,
  email VARCHAR(255) NOT NULL UNIQUE,
  phone VARCHAR(30),
  dob DATE,
  gender VARCHAR(20),
  country VARCHAR(100),
  city VARCHAR(100),
  password_hash TEXT NOT NULL,
  role_id UUID REFERENCES role(id),
  is_active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT now(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- books
CREATE TABLE book (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title VARCHAR(500) NOT NULL,
  author VARCHAR(255),
  isbn VARCHAR(50),
  description TEXT,
  cover_url TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- user_book (many-to-many)
CREATE TABLE user_book (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES "user"(id) ON DELETE CASCADE,
  book_id UUID NOT NULL REFERENCES book(id) ON DELETE CASCADE,
  read_date DATE,
  status VARCHAR(30) DEFAULT 'not_started',
  rating INT CHECK (rating BETWEEN 1 AND 5),
  notes TEXT,
  UNIQUE(user_id, book_id)
);

-- documents (pdfs/structured data)
CREATE TABLE document (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title VARCHAR(500) NOT NULL,
  description TEXT,
  file_url TEXT NOT NULL,
  file_type VARCHAR(50),
  uploaded_by UUID REFERENCES "user"(id),
  uploaded_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- document tags
CREATE TABLE document_tag (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  document_id UUID REFERENCES document(id) ON DELETE CASCADE,
  tag VARCHAR(150)
);

-- bot model versions
CREATE TABLE bot_model (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name VARCHAR(200),
  config JSONB,
  deployed_at TIMESTAMP WITH TIME ZONE
);

-- chat sessions
CREATE TABLE chat_session (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES "user"(id) ON DELETE SET NULL,
  session_type VARCHAR(30) DEFAULT 'text',
  bot_model_id UUID REFERENCES bot_model(id),
  started_at TIMESTAMP WITH TIME ZONE DEFAULT now(),
  ended_at TIMESTAMP WITH TIME ZONE
);

-- messages
CREATE TABLE message (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  chat_session_id UUID REFERENCES chat_session(id) ON DELETE CASCADE,
  sender_id UUID, -- NULL for system/bot (or use special id)
  sender_type VARCHAR(20) DEFAULT 'user', -- 'user' | 'bot' | 'system'
  message_type VARCHAR(20) DEFAULT 'text', -- 'text' | 'audio' | 'file' | 'system'
  content TEXT,
  media_url TEXT,
  duration_seconds INT,
  sent_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- attachments
CREATE TABLE attachment (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  message_id UUID REFERENCES message(id) ON DELETE CASCADE,
  file_url TEXT,
  file_name VARCHAR(500),
  mime_type VARCHAR(100),
  file_size BIGINT
);
